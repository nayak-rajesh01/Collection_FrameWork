# 📚 Concepts Covered

### 📋 Java List

Ordered collection that allows **duplicate elements**.

Methods practiced:

* `add()`
* `get()`
* `set()`
* `remove()`
* `size()`

---

### 📦 ArrayList

Features:

* Dynamic array
* Maintains insertion order
* Allows duplicates
* Fast random access

Example:

```java
ArrayList<String> list = new ArrayList<>();

list.add("Java");
list.add("Python");
list.add("C++");
```

---

### 🔗 LinkedList

Features:

* Doubly linked list structure
* Efficient insertion and deletion
* Slower random access compared to ArrayList

---

### 🔄 List Sorting

Java provides sorting utilities through the `Collections` class.

Examples:

```java
Collections.sort(list);
Collections.reverse(list);
Collections.shuffle(list);
```

---

# 🧮 Java Set

A **Set** does not allow duplicate elements.

Types learned:

### ⚡ HashSet

* Uses hashing
* No duplicate elements
* No guaranteed order

### 🌳 TreeSet

* Stores elements in **sorted order**
* Uses **Red-Black Tree**

### 🔗 LinkedHashSet

* Maintains **insertion order**

---

# 🗂 Java Map

Maps store data as **key-value pairs**.

Example structure:

```
Key → Value
101 → Rajesh
102 → Rahul
```

Types learned:

### ⚡ HashMap

* Fast performance
* Allows one null key
* No order guarantee

### 🌳 TreeMap

* Keys stored in **sorted order**

### 🔗 LinkedHashMap

* Maintains insertion order of keys

---

# 🔁 Java Iterator

Used to **traverse collections**.

Example:

```java
Iterator<String> it = list.iterator();

while(it.hasNext()) {
    System.out.println(it.next());
}
```
